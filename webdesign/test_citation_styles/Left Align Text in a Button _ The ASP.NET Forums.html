    Left Align Text in a Button

    Aug 01, 2013 04:58 PM|SeanGrebey|LINK

    I need to dynamically create some buttons on my page. I would really like the text to be aligned left instead of centered. Any thoughts on how to do this? I thought this would work but no dice:

    ASPX

    <head runat="server">
    <title></title>
    <style type="text/css">
    
    stbutton
    {
        text-align:left;
        
    }
    </style>
</head>

    Code Behind:

        Protected Sub Page_Init(ByVal sender As Object, ByVal e As System.EventArgs) Handles Me.Init
        Dim TestButton1 As New Button

        TestButton1.Width = 150
        TestButton1.Height = 150
        TestButton1.BackColor = Drawing.Color.Red
        TestButton1.BorderColor = Drawing.Color.Black
        TestButton1.BorderStyle = BorderStyle.Solid
        TestButton1.BorderWidth = 5
        AddHandler TestButton1.Click, AddressOf buttonClick

        Dim sb As New System.Text.StringBuilder
        sb.AppendLine("PO: 12345")
        sb.AppendLine("Status: Open")
        TestButton1.Text = sb.ToString
        TestButton1.CssClass = "stbutton"

        Panel1.Controls.Add(TestButton1)


    End Sub

    Re: Left Align Text in a Button

    Aug 01, 2013 05:03 PM|Rion Williams|LINK

    You'll need your CSS style to actually target a class of stbutton, so you'll need to preface the style with a dot '.' to denote classes : 

    <style type="text/css">
    .stbutton{
        text-align:left;
    }
</style>
